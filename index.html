    Cloud Manager のドキュメント

    Cloud Manager provides IT experts and cloud architects with a centralized control plane to manage, monitor, and automate data in hybrid cloud environments. Browse the docs by product area or use the search above to find content anywhere in the Cloud Manager docs.

    すべてのコンテンツ
    セットアップと管理

    Cloud Manager の使用を開始し、コネクタやアカウントなどを管理する方法について説明します。

    リリースノート

    Cloud Manager の新機能をご確認ください。

    Active IQ デジタルアドバイザ

    実用的な情報でストレージの健常性を最適化

    ONTAP 対応の Amazon FSX

    AWS が管理する標準サービスとして、 ONTAP の全機能を使用できます。

    API

    クラウドベースのリソースをプログラムで管理できます。

    AppTemplate （アプリケーションテンプレート）

    Cloud Manager でアクションを自動化することで、プロビジョニングを高速化し、手動作業を排除できます。

    Azure NetApp Files の特長

    コアビジネスアプリケーション向けのハイパフォーマンスなファイルストレージサービス。

    クラウドバックアップ

    クラウドとオンプレミスのデータをバックアップ、リストア

    クラウドデータの意味

    アプリケーションデータとクラウド環境のプライバシーを確保

    Cloud Sync

    オンプレミスとクラウドのデータストア間でデータを同期

    クラウド階層化

    オンプレミスの ONTAP ストレージをクラウドに拡張

    Cloud Volumes ONTAP

    クラウドストレージを低コストでインテリジェントに、コンプライアンスに準拠し、セキュアに構築できます。

    Cloud Volumes Service for AWS

    オールフラッシュのパフォーマンスを発揮するクラウドネイティブなファイルサービスです。

    Cloud Volumes Service for Google Cloud

    ファイルストレージをセキュアに、保護、高速化します。

    コンピューティング

    クラウドコンピューティング関連の支出を可視化し、削減可能なコストを特定できます。

    グローバルファイルキャッシュ

    分散した従業員にリアルタイムのグローバルファイル共有を提供します。

    Kubernetes クラスタ

    ハイブリッドクラウド内の Kubernetes クラスタの高度なデータ管理。

    監視

    Cloud Volumes ONTAP の健全性とパフォーマンスを把握できます。

    オンプレミスの ONTAP クラスタ

    ハイブリッドクラウド全体にわたって NetApp AFF システムと FAS システムを管理できます。

    ランサムウェアからの保護

    サイバー攻撃に対するデータの復元力を評価します。

    レプリケーション

    バックアップとディザスタリカバリをサポートするために、ストレージシステム間でデータをレプリケートします。

    SnapCenter サービス

    クラウドでのアプリケーションと整合性のあるデータ保護
